1. 克隆代码
 git clone +网址

 ctrl + insert  复制
 shift + insert 粘贴

 2. 删除分支
 git branch -D 分支名

 3. 查看分支
 git branch +分支名

 4. 创建分支
 git checkout -b 分支名

 5. 切换分支
 git checkout 分支名

 6. 查看状态
 git status

 7. 添加修改的数据到缓存区
 git add .
 git add 修改的文件名

 8. 提交修改到本地分支
 git commit -m ‘注解’

 9. 提交本地分支到远程分支上
 git push origin 分支名

 10. 设置全局变量
 git config --global user.email "sss@qq.com"

 git config --global user.name "69"

 11. 合并add和commit操作
 git commit -am ‘’

 12. 创建秘钥
 ssh-keygen -t rsa -C 账号

 13. 比较分支之间的不同
 git diff 分支1 分支2

 14. 合并
 git merge 分支

 15. 打tag标签
 git tag -a 版本号 -m 注解

 16. 推送版本
 git push origin 版本号

 17. 删除远程分支
git push origin --delete 分支

 18. 查看版本号
 git tag 

 19. 删除版本号
 git tag -d 版本号

 20. 删除远程版本号
 push origin --delete tag 版本号
 
 21. 缓存修改的代码
 git stash

 22. 查看缓存
 git stash list

 23.还原缓存
 git stash apply stash@{x}
